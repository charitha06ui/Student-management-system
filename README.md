# Student Management System (Python)

## 📌 Project Description
This is a simple Student Management System built using Python. It uses file handling to store student data permanently in a text file.

---

## 🚀 Features
- Add Student
- View Students
- Search Student
- Delete Student
- Update Student
- Save data permanently in file (students.txt)

---

## 🛠️ Technologies Used
- Python
- File Handling
